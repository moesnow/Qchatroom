# Qchatroom

基于Qt开发的多人聊天室

## 编译

#### Linux/macOS

```bash
git clone https://github.com/moesnow/Qchatroom.git
cd Qchatroom
./build
```

## 用法

#### macOS

```bash
open server/server.app
```

```bash
open -n client/client.app
```

#### Linux

```bash
./server/server
```

```bash
./client/client
```
